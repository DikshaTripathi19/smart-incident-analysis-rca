# Smart Incident Analysis RCA

Automated **Root Cause Analysis (RCA)** tool using **Java Spring Boot** and **Large Language Models (LLM)**.  
This application provides AI-driven insights into incidents, including potential root cause, impact, and resolution steps.

## Features

- AI-powered RCA for incidents in **production, staging, and QA environments**
- Real-time analysis based on **incident description, severity, environment, and symptoms**
- Stores incident data in **PostgreSQL**
- Interactive **Swagger API documentation**
- Modern **Java 21** with **records** for immutable data transfer
- Extensible for new incident types or AI models

## Tech Stack

- **Backend:** Java 21, Spring Boot  
- **Database:** PostgreSQL  
- **API Documentation:** Swagger / OpenAPI  
- **AI Integration:** OpenAI GPT / LLM  
- **Build Tool:** Maven (Maven Wrapper included)
